#### Installation

Clone repo

```bash
$ git clone https://github.com/IrinaEnotova/node-crud-api.git
```

Change directory

```bash
$ cd node-crud-api
```

Change branch

```bash
$ git checkout dev
```

Install dependencies

```bash
$ npm install
```

#### Running the app

Start development server:

```
$ npm run start:dev
```

Start production server:

```
$ npm run start:prod
```

Start server with Cluster API:

```
$ npm run start:multi
```

Run tests:

```
$ npm run test
```
